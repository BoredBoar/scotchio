# Getting started with-Angular-1.5.x components
## Introduction

Web Components are a set of features that allow for the creation of reusable widgets or components in web documents and web applications. The components model allows for encapsulation and interoperability of individual HTML elements. Previously in angular  you could build a components by using `.directive()` method helper, with this method you could build custom HTML elements and attribute giving your application the modularity and encapsulation of DOM elements.
